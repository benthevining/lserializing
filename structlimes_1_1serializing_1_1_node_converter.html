Detailed Description

template<typename Type>
struct limes::serializing::NodeConverter< Type >

This template class is used to convert specific types to and from Node objects.

In order to compose types for serialization, they either need to inherit from SerializableData or there needs to be a specialization of this template for your type.

This is a template interface, not a base clas. You should specialize this template for your type; see the specialization for the ImplementsSerializableData concept for an example.
