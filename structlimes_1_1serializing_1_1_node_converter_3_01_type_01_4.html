Detailed Description

template<enums::Enum Type>
struct limes::serializing::NodeConverter< Type >

A specialization of NodeConverter for any enum type.

A template specialization for NodeConverter that serializes and deserializes any SerializableData object using the SerializableData API.

Internally, Node objects store numbers as doubles.
