Detailed Description

This class manages a registry of Format objects.

The following serialization formats are provided by this library:

  • JSON
  • XML
  • YAML
  • TOML
  • INI

Other third party libraries may add new serialization formats and register them with the KnownFormats object, allowing Limes code to find and use custom formats as if they were native to Limes.
